Waterpolo Hub Quickstart (3 minutes)

Set up and start tracking your first match quickly.

1. Create workspace

  1. Create season and team.
  2. Add your roster with cap numbers.
  3. Create your first match in Matches.

2. Track live

  1. Open Scoring and select match.
  2. Use live controls for period and clock.
  3. Tap player + action to log each event.

3. Analyze and report

  1. Open Stat Sheet for match and season summaries.
  2. Use Shotmap + Analytics for zone and distance trends.
  3. Export CSV/PDF and backup bundle from Settings.
